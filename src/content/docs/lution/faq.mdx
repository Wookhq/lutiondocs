### Does Bloxstrap FFlags work on Lution?
Bloxstrap FFlags **should work on Lution (Sober)** as long as the FFlag isn't exclusive to the Windows platform. Some FFlags are designed to only work on Windows and are not compatible with Android.

***
### Will Lution work on Linux Mint?
Yes, Lution **will work on Linux Mint** as long as you have **Python** installed and follow the official documentation for setup. Lution is designed to be compatible with most Linux distributions.

***
### What if one day I get my Roblox account stolen by using Lution?
Lution is an **open-source project**, meaning its source code is publicly available for anyone to review. This transparency allows the community to audit the code for any malicious or insecure practices. While it's always important to be cautious when using third-party software, the open-source nature of Lution provides a level of assurance because the code is not hidden. Additionally, Lution is designed to work exclusively on the Linux operating system.
